> **"eCAMS AI는 단순한 챗봇이 아닙니다. 당신의 시스템을 가장 잘 아는 시니어 유지보수 엔지니어입니다."**

eCAMS AI는 복잡한 소스코드와 데이터베이스 구조를 학습하여, 장애 분석, 로직 파악, 영향도 분석 등을 한국어로 즉시 답변해 주는 인공지능 분석 플랫폼입니다.

---

## 1. 시작하기 (Getting Started)

### 🔐 회원가입 및 로그인
1. **가입 신청**: 아이디, 비밀번호, 성명, 소속을 입력하여 가입을 신청합니다.
2. **관리자 승인**: 가입 신청 후 관리자의 승인이 있어야 로그인이 가능합니다.
3. **보안 정책**: 5회 이상 로그인 실패 시 IP가 15분간 차단되니 주의하세요.

### 🔑 권한 신청
- 로그인 후 사이드바의 **[권한 신청]** 메뉴를 통해 분석이 필요한 레포지토리의 접근 권한을 요청할 수 있습니다.
- 권한은 **읽기(Read)**와 **수정(Edit)**으로 구분됩니다.

---

## 2. 레포지토리 관리 (Repository Management)

### 📁 새 레포지토리 생성
관리자 또는 권한이 있는 사용자는 소스코드를 시스템에 등록할 수 있습니다.
- **ZIP 업로드**: 로컬의 소스코드를 압축하여 업로드합니다.
- **Git Clone**: 원격 저장소 URL을 통해 소스코드를 직접 가져옵니다.

> [!TIP]
> **고객사별 자동 분류**: 레포지토리 생성 시 고객사를 선택하면 `workspace/[고객사명]/[레포명]` 구조로 깔끔하게 정리됩니다.

### 📑 코드 인덱싱
- 신규 레포지토리 생성 시 AI가 코드를 이해하기 위한 **인덱스 빌드**가 백그라운드에서 진행됩니다.
- 인덱싱이 완료되면 AI의 답변 속도와 정확도가 비약적으로 향상됩니다.

---

## 3. 스마트 분석 기능 (AI Analysis)

### 💬 AI와 대화하기
- **한국어 질의응답**: "이 프로시저에서 CMR1800 테이블을 사용하는 로직이 어디야?"와 같이 구체적으로 질문하세요.
- **모델 선택**: 
  - ✨ **Gemini**: 가벼운 질문이나 빠른 확인이 필요할 때 적합합니다.
  - ⚡ **Sonnet**: 복잡한 로직 분석이나 정밀한 코드 검토가 필요할 때 사용하세요.

### 📸 사진 첨부 분석
- 에러 화면 스크린샷이나 시스템 UI 사진을 첨부하면, AI가 화면 구성을 분석하거나 에러 메시지를 파악하여 관련 코드를 찾아줍니다.

### 💻 소스 뷰어 (Source Viewer)
- 사이드바의 **[소스 뷰어]**를 통해 소스코드를 직접 탐색할 수 있습니다.
- **인코딩 지원**: UTF-8 뿐만 아니라 EUC-KR 파일도 깨짐 없이 볼 수 있습니다.
- **다운로드**: 분석 중인 파일을 로컬로 즉시 다운로드할 수 있습니다.

## 4. 지능형 학습 시스템 (Knowledge Learning)

eCAMS AI는 사용자의 피드백을 통해 점점 더 똑똑해집니다.

1. **지식 저장 (좋아요 👍)**: AI의 답변이 유익했다면 좋아요를 누르세요. 해당 답변은 **벡터 DB**에 저장되어, 나중에 비슷한 질문이 들어오면 AI가 다시 검색하지 않고 즉시 답변합니다.
2. **오류 수정 (싫어요 👎)**: 답변이 틀렸다면 싫어요를 누르고 수정 내용을 입력해 주세요. 잘못된 지식은 캐시에서 삭제되고 새로운 지식으로 업데이트됩니다.

---

## 5. 관리자 도구 (Admin Tools)

### 🛡️ 결재함 관리
- 신규 가입자 승인 및 레포지토리 권한 신청 건을 검토하고 승인합니다.

### 👥 사용자 및 고객사 관리
- 사용자의 권한을 개별적으로 수정하거나 고객사별 자산을 그룹화하여 관리할 수 있습니다.

### 📑 인덱스 상태 모니터링
- 모든 레포지토리의 인덱스 생성 상태를 확인하고, 필요한 경우 인덱스를 다시 빌드할 수 있습니다.
